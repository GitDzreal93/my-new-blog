标准库¶

  1. os 负责程序与操作系统的交互，提供了访问操作系统底层的接口
  2. sys 提供访问由解释器使用或维护的变量和在与解释器交互使用到的函数

  4. datetime 日期
  5. subprocess 命令行交互
  6. math 数学
  7. threading 多线程
  8. multiprocessing 多进程
  9. asyncio 异步io
  10. re 正则
  11. collections 数据结构
  12. abc 抽象类
  13. codecs 编码
  14. calendar 日历
  15. copy 拷贝
  16. enum 枚举

  20. hashlib 哈希函数
  21. argparse 命令行选项、参数
  22. getopt 命令行选项、参数
  23. logging 日志
  24. concurrent.futures 优雅地写多线程、多进程
  25. queue 队列
  26. socket socket网络编程
  27. json json库
  28. webbrowser 浏览器
  29. tkinter 图形界面
